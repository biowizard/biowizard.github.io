--- layout: mdpage --- # Getting Started 1. Launch an EC2 instance with the trial Biowizard AMI Biowizard is distributed as part of an Amazon Machine Image (AMI). Users can start up an Amazon EC2 instance with Biowizard and all the tools it uses baked right in. The current trial version of Biowizard is bundled in AMI ```ami-729a501a```. 1. Download an example job file to your instance ``` ssh -i mykey ubuntu@myinstance.amazonaws.com "wget http://biowizard.github.io/examples/map.yml" ``` 1. Log in to your instance and run your job ``` ssh -i mykey ubuntu@myinstance.amazonaws.com "biowizard map.yml" ```
